# Expo 앱에 오신 것을 환영합니다 👋

이 프로젝트는 [`create-expo-app`](https://www.npmjs.com/package/create-expo-app)으로 생성된 [Expo](https://expo.dev) 프로젝트입니다.

## 시작하기

1. 의존성 패키지 설치하기

   ```bash
   npm install
   ```

2. 앱 시작하기

   ```bash
   npx expo start
   ```

터미널 출력 화면에서 앱을 실행할 수 있는 다음과 같은 옵션들을 확인할 수 있습니다.

- [개발 빌드 (development build)](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android 에뮬레이터](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS 시뮬레이터](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go) (Expo로 앱 개발을 가볍게 시작하고 테스트해 볼 수 있는 제한된 샌드박스 환경)

**app** 디렉토리 안의 파일들을 수정하여 개발을 시작할 수 있습니다. 이 프로젝트는 [파일 기반 라우팅](https://docs.expo.dev/router/introduction)을 사용합니다.

## 새로운 프로젝트로 초기화하기

기존 예제 코드를 지우고 새로 시작할 준비가 되었다면 다음 명령어를 실행하세요:

```bash
npm run reset-project
```

이 명령어는 스타터 코드를 **app-example** 디렉토리로 이동시키고, 개발을 시작할 수 있는 빈 **app** 디렉토리를 생성합니다.

### 기타 설정 단계

- ESLint를 설정하여 코드 스타일을 검사하려면 `npx expo lint`를 실행하거나, ["ESLint 및 Prettier 사용 가이드"](https://docs.expo.dev/guides/using-eslint/)를 참고하세요.
- 단위 테스트(unit testing)를 설정하려면 ["Jest를 이용한 단위 테스트 가이드"](https://docs.expo.dev/develop/unit-testing/)를 참고하세요.
- 이 템플릿의 TypeScript 설정에 대해 더 자세히 알아보려면 ["TypeScript 사용 가이드"](https://docs.expo.dev/guides/typescript/)를 참고하세요.

## 더 알아보기

Expo를 이용한 프로젝트 개발에 대해 더 자세히 알아보려면 다음 자료를 참고하세요:

- [Expo 공식 문서](https://docs.expo.dev/): 기본 개념부터 [가이드](https://docs.expo.dev/guides)를 통한 고급 주제까지 다룹니다.
- [Expo 학습 튜토리얼](https://docs.expo.dev/tutorial/introduction/): Android, iOS, 웹에서 작동하는 프로젝트를 직접 만들어보는 단계별 튜토리얼입니다.
